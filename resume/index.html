I led a team of three, and we built a web application that helps businesses summarize their financials for an at-a-glance view of financial health. Users upload Quickbooks data files and then categorize accounts into predefined “buckets”. The application can then analyze these buckets and report back to the user. It was built with Symfony2, Backbone, and MongoDB.
